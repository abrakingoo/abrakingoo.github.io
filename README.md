## ğŸ¨ Customization

### Change Theme Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --accent-primary: #6366f1;
    --accent-secondary: #8b5cf6;
    /* ... */
}
```

### Add New Projects
1. Add project HTML in `index.html`
2. Add project image to `img/` folder
3. Update CSS with new image class in `styles.css`

### Modify Content
- **Personal Info:** Edit sidebar in `index.html`
- **Skills:** Update skills section in `index.html`
- **Projects:** Add/edit project cards in `index.html`
